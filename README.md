# CarePlus - Pharmacy Management System

## Tech Stack
Java, JSP, Servlets, Spring MVC, MySQL

## Features
- Inventory Management
- Billing System
- Supplier Management
- Analytics Dashboard

## Description
This is a full-stack pharmacy management system developed to manage inventory, billing, and supplier operations efficiently.

## How to Run
1. Import project into Eclipse/IntelliJ
2. Configure Tomcat Server
3. Connect MySQL database
4. Run the project
